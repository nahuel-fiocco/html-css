SEO:

	Descripcion:    Con la etiqueta Meta name description agregue el sgte parrafo que resume el contenido y objetivo del sitio: "Un blog dedicado a quienes quieran turistear por Buenos Aires, lleno de ideas para salir a pasear, comer, sacar fotos y disfrutar de la ciudad de la furia como nunca antes la viviste".

	Keywords:       Mediante la etiqueta Meta name keywords configure las palabras clave que use en mi sitio: "turismo, buenos aires, turistear, turisteando, fooding, escapadas, salidas, paseos, turistas, fotografia, foodporn, nahuel, fiocco, micka, sofia, food, blog".

	Encabezado:     Los encabezados son diferentes en cada sitio, siendo autodescriptivos en el sitio que se encuentren: "TuristeandoBA | Home/Turistear/Fooding/Nosotros"
